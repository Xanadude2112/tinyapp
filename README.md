# TinyApp Project

## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install <the dev dependency you wish to install>` command).
- Run the development web server using the `node express_server.js` command.
